# Results

This directory contains benchmark outputs, evaluation metrics, visualizations, and experimental results for the Online Feedback Distillation framework.

## Evaluation Metrics

* BERTScore
* ROUGE-L
* BLEU
* Cosine Similarity
* Toxicity (Detoxify)
* Exact Match Accuracy
* LLM-as-Judge Evaluation

## Benchmarks

* GSM8K
* Alpaca
* CommonGen

## Contents

* `figures/` - plots and visualizations
* `tables/` - benchmark result tables
* `metrics/` - exported evaluation metrics
* `ablations/` - ablation study results
